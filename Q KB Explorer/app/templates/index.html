Help & Documentation

About Q KB Explorer

An offline-capable explorer for Qualys Knowledge Base (QIDs), Compliance Controls (CIDs), Policy Compliance Policies, and Compliance Mandates. Built by netsecops-76 as an independent project.

Disclaimer: This tool is not affiliated with, endorsed by, or supported by Qualys, Inc.

Quick Start

  • Go to Settings and select your Qualys platform, then enter your credentials.
  • Click Test Connection to verify access.
  • Click Save Settings & Credential to store securely in the AES-256 vault.
  • Use the Data Sync section to pull QIDs, CIDs, Policies, and Mandates.
  • Search and explore data in the QIDs, CIDs, Policies, and Mandates tabs.

Data Types

  • QIDs (Knowledge Base) — Vulnerability detections with severity, CVEs, CVSS scores, diagnosis, and solution information.
  • CIDs (Compliance Controls) — Individual compliance checks with criticality ratings, technology associations, and linked policies.
  • Policies — Collections of compliance controls that can be exported, imported, and migrated between Qualys environments.
  • Mandates (Frameworks) — Regulatory frameworks (PCI DSS, HIPAA, etc.) that map to compliance controls.

Search & Filters

  • Full-text search — Type keywords to search across titles, descriptions, and metadata. Supports prefix matching for partial words.
  • Multi-select filters — Use dropdowns to filter by severity, category, technology, etc. Combine with AND/OR toggle.
  • CVE search — Type-ahead search for CVE IDs with debounced server queries.
  • Advanced filters — Expand for CVSS ranges, date ranges, PCI flag, vulnerability type, and threat intelligence indicators.
  • Recent searches — Click the clock icon next to any search bar to see and re-run recent searches.

Policy Migration

  • Export — From a policy detail view, click Export to download the full policy XML from Qualys and store locally.
  • Import — Use the Migration tab to import policy XML files from local upload.
  • Upload — Select a destination credential and upload stored policies to another Qualys environment.
  • Stale detection — Policies modified after export are flagged for re-export.

Bookmarks

  • Click the star icon on any QID, CID, or Policy card to bookmark it.
  • Press b while viewing a detail modal to toggle its bookmark.
  • Bookmarks are stored locally in your browser (up to 500 items).

Bulk Export

  • Click the Select button in the QID or CID tab header to enter select mode.
  • Check individual items or use Select All, then export as CSV.
  • Bulk exports include full detail fields (CVEs, diagnosis, solution, etc.).
  • CSV exports are unlimited across all tabs.

Data Sync

  • QIDs and CIDs sync independently from the Qualys API.
  • Policies require CIDs to be synced first — policies reference CID controls for technology filters and cross-references.
  • Mandates (Frameworks) are automatically extracted during CID sync — no separate sync needed. Mandate data comes from FRAMEWORK_LIST elements in the CID API response.
  • Delta Sync — Pulls only records modified since last sync. Fast and efficient.
  • Full Sync — Pulls all records. Use for initial sync or monthly refresh. Existing data is purged first.
  • Scheduling — Set up recurring syncs (weekly, bi-weekly, monthly) with timezone support.
  • Data is cached locally in SQLite for fast offline search.

Keyboard Shortcuts

KeyAction

Troubleshooting

  • Empty results after sync — Ensure your credential has API access. Check the sync log for errors.
  • Session expired — Re-enter your password in the vault authentication dialog.
  • Sync timeout — Large datasets (100K+ QIDs) may take 10-15 minutes. Keep the browser tab open.
  • Missing mandates — Mandates are extracted from CID sync data. Run a CID sync first.
