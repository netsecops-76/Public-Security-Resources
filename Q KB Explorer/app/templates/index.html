Help & Documentation

About Q KB Explorer

A local, Docker-based tool for exploring the Qualys Knowledge Base (QIDs), Compliance Controls (CIDs), Policy Compliance, Asset Tags, and Patch Management data. Features full-text search, cross-referencing, threat intelligence, and cross-environment migration. Built by netsecops-76.

Disclaimer: This tool is not affiliated with, endorsed by, or supported by Qualys, Inc.

Quick Start

  • Go to Settings and select your Qualys platform, then enter your credentials.
  • Click Test Connection to verify access.
  • Click Save Settings & Credential to store securely in the AES-256 vault.
  • Use the Data Sync section to pull QIDs, CIDs, Policies, Tags, and PM Patches.
  • Search and explore data in the QIDs, CIDs, Policies, Mandates, Intelligence, and Tags tabs.

Data Types

  • QIDs (Knowledge Base) — 200K+ vulnerability detections with severity, CVEs, CVSS scores, threat intelligence (active attacks, CISA KEV, exploits, malware), diagnosis, solution, and linked PM patches.
  • CIDs (Compliance Controls) — Individual compliance checks with criticality ratings, technology associations, and linked policies.
  • Policies — Collections of compliance controls that can be exported, imported, and migrated between Qualys environments.
  • Mandates (Frameworks) — Regulatory frameworks (PCI DSS, HIPAA, etc.) that map to compliance controls.
  • Tags — Asset tags with parent-child hierarchy, rule types, and origin classification. Full CRUD, migration, and audit capabilities.
  • PM Patches — 218K+ Windows and Linux patches from the Qualys Patch Management catalog, linked to QIDs.

Intelligence Tab

  • Threat filters — Active Attacks, CISA KEV, Public Exploit, RCE, Malware, Has Exploits. Click to filter; click a second time on the active filter bar to negate (NOT).
  • Clickable metrics — Click any stat card to drill into that subset of your data.
  • OR logic — Multiple filters in the same category (e.g. PM Win + PM Lin) return the union, not intersection.
  • Save/Load searches — Save named filter combinations for reuse. Click Saved to load a previous search.
  • Active filter bar — Shows all applied filters as removable tags. Click a tag to toggle NOT (exclude). Click X to remove.

Tags

  • Browse — Parent-child tree view with expand-on-click. Flat view when filters are active. Shows child count and parent reference on every card.
  • Library — 136 built-in tag templates grouped by rule type. Click to view details, Clone to edit, Apply to push to a Qualys environment.
  • Audit — Checks for hierarchy issues, duplicate names, duplicate rules, and naming problems. Run before migration to catch issues.
  • Migration — Export/import tag JSON files, or migrate directly between subscriptions.
  • Select mode — Click Select, then choose individual tags or "Select all". Use "+ children" on parent cards to select recursively. Actions: Migrate to env, Export to JSON, Import JSON, Delete local, Delete from Qualys.
  • Origin badges — Tags are classified as rule_based (has detection logic), static (no rule, grouping), connector (cloud connector dependent), or system (Qualys-provisioned, cannot migrate/delete).

Tag Migration

  • Origin picker — Choose which tag types to include: rule-based (default on), static (default on), connector (default off, needs matching connector), system (always excluded).
  • Collision detection — Before migrating, the tool checks the destination for existing tag names. Per-tag options: Rename (editable), Skip, Skip all, Rename all.
  • Async with progress — Migration runs in the background with a live progress bar. Results persist even if you close the modal.
  • Provenance — Tags track which subscription they came from. Migrating back to the same subscription updates; migrating to a different one creates new.

Search & Filters

  • Full-text search — Type keywords to search across titles, descriptions, and metadata. Supports prefix matching for partial words.
  • Multi-select filters — Use dropdowns to filter by severity, category, technology, rule type, etc.
  • CVE search — Type-ahead search for CVE IDs with debounced server queries.
  • Advanced filters — Expand for CVSS ranges, date ranges, PCI flag, vulnerability type, and threat intelligence indicators.
  • Recent searches — Click the clock icon next to any search bar to see and re-run recent searches.
  • Clear button — Resets all filters and search text. Located next to the Search button.

Policy Migration

  • Export — From a policy detail view, click Export to download the full policy XML from Qualys and store locally.
  • Import — Use the Migration tab to import policy XML files from local upload.
  • Upload — Select a destination credential and upload stored policies to another Qualys environment.
  • Stale detection — Policies modified after export are flagged for re-export.

Data Sync

  • QIDs — 208K+ vulnerabilities with threat intelligence. Full sync in ~9 minutes.
  • CIDs — 26K+ compliance controls. Policies require CIDs to be synced first.
  • Tags — Asset tags via QPS REST API. Includes parent-child relationships.
  • PM Patches — 218K+ patches (Windows + Linux) via Gateway JWT API.
  • Mandates are automatically extracted during CID sync.
  • Delta Sync — Pulls only records modified since last sync. Typically 2 seconds.
  • Full Sync — Pulls all records. Use for initial sync or monthly refresh.
  • Scheduling — Set up recurring delta syncs. Schedule badges show next run time.
  • Backfill Missing — Recovers gaps from interrupted syncs without re-downloading everything.

Dashboard

  • Data Inventory — Record counts and sync status for all 6 data types.
  • Threat Intelligence — Active Attacks, CISA KEV, Public Exploits, RCE, Has Exploits counts.
  • Charts — QID severity distribution, CID criticality, top 15 categories, patchable percentage.
  • Sync Health — Last sync times and health status for all data types.

Updating

  • Check for Updates — Settings tab, compares your version against the public repo.
  • Apply Update — Downloads new code, installs dependencies, restarts workers. No container rebuild needed.
  • Your data is always safe — Updates only replace application code. All synced data lives on a separate Docker volume.
  • If something breaks — Run docker compose build --no-cache && docker compose up -d to restore.

Bookmarks & Bulk Export

  • Click the star icon on any card to bookmark it. Press b in a detail modal to toggle.
  • Click Select in any tab to enter select mode for bulk CSV export.
  • Tags support additional bulk actions: Migrate, Export JSON, Delete local, Delete from Qualys.

Keyboard Shortcuts

Troubleshooting

  • Empty results after sync — Ensure your credential has API access. Check the sync log (click "Last Sync Details").
  • Session expired — Re-enter your password in the vault authentication dialog.
  • Sync timeout — QIDs take ~9 min, PM Patches ~3 min. Keep the browser tab open.
  • Missing mandates — Mandates are extracted from CID sync data. Run a CID sync first.
  • Tags showing as SYSTEM incorrectly — Re-sync tags. The classification runs on every sync.
  • PM Linux patches showing 0 — Update to v2.1.0+. Previous versions had a filter bug.
  • Update broke the app — Run docker compose build --no-cache && docker compose up -d. Data is safe.
