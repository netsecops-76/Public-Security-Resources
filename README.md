# Qualys Cloud Agent Log Viewer

A single-file HTML tool for parsing and analyzing Qualys Cloud Agent log files and CAR (Custom Assessment & Remediation) job report CSVs. Drop files into your browser and get an organized, searchable, color-coded view instantly.

## Getting Started

See the full documentation and source in the [Qualys Cloud Agent Log Viewer](Qualys%20Cloud%20Agent%20Log%20Viewer/) folder.

## Quick Start

1. Download `qualys-log-viewer.html` from the [Qualys Cloud Agent Log Viewer](Qualys%20Cloud%20Agent%20Log%20Viewer/) folder
2. Open it in any modern browser (Chrome 105+, Edge 105+, Firefox 113+, Safari 16.4+)
3. Drag and drop your log files or CAR report CSVs onto the upload zone

No server, no install, no build step required.

## Supported Formats

`.log`, `.csv` (CAR job reports), `.zip`, `.tar.gz`, `.tgz`, `.gz`, `.7z`, and folders — including Qualys REMOTELOG bundles from both Linux and Windows agents.
