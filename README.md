# Public Security Resources

A collection of security scripts and tools available for use at your own risk.

## Available Resources

| Resource | Description | Platforms | Branch |
|----------|-------------|-----------|--------|
| [Qualys Cloud Agent Troubleshooting Scripts](https://github.com/netsecops-76/Public-Security-Resources/tree/Qualys-Cloud-Agent-Troubleshooting-Scripts/Qualys%20Cloud%20Agent%20Troubleshooting%20Scripts) | Automated diagnostic and troubleshooting tools for the Qualys Cloud Agent. | [Q KB Explorer](https://github.com/netsecops-76/Public-Security-Resources/tree/Q-KB-Explorer/Q%20KB%20Explorer) | Qualys Knowledge Base and Policy Compliance explorer with local caching, full-text search, cross-referencing, and cross-environment policy migration. | [Qualys Cloud Agent Log Viewer](https://github.com/netsecops-76/Public-Security-Resources/tree/Qualys-Cloud-Agent-Log-Viewer/Qualys%20Cloud%20Agent%20Log%20Viewer) | Single-file HTML tool for parsing and analyzing Qualys Cloud Agent log files and CAR job report CSVs. Smart grouping, search, host detail extraction, and redaction. Supports REMOTELOG bundles, archives, and multiple log types. | Browser (HTML) | `Qualys-Cloud-Agent-Log-Viewer` |
| [Qualys CAR Scripts](https://github.com/netsecops-76/Public-Security-Resources/tree/Qualys-CAR-Scripts/Qualys%20CAR%20Scripts) | Parameterized scripts for Qualys Custom Assessment and Remediation (CAR). Audit-first remediation plus read-only assessment scripts: database inventory, local admin create/remove, BigFix and Okta uninstall. | Windows (PowerShell 5.1), Linux (Bash) | `Qualys-CAR-Scripts` |
